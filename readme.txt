1) Crea il virtual environment “envForGithubActions” 
2) Configura le variabili d'ambiente, seguendo e customizzando il seguente esempio:

DIR_REPORT_TRADIZIONALI: /home/runner/work/Tesi-ReportFinali/Tesi-ReportFinali/Tradizionale
DIR_REPORT_HOOKS: /home/runner/work/Tesi-ReportFinali/Tesi-ReportFinali/Hooks
DIR_BASE_PATH: /home/runner/work/Tesi-ReportFinali/Tesi-ReportFinali
EMAIL_ACCOUNT_GITHUB: t*********@gmail.com
NOME_ACCOUNT_GITHUB: ********
PASSWORD_ACCOUNT_GITHUB: ***********
NOME_REPORT_TIPO_1: reportTradizionaliComplessivo
NOME_REPORT_TIPO_2: reportHooksComplessivo

Successivamente:

3) Inserisci all'interno della directory Tradizionale, i report (xls) dei test Tradizionali eseguiti, ottenuti dal progetto principale
4) Inserisci all'interno della directory Hooks, i report (xls) dei test con gli hooks eseguiti, ottenuti dal progetto principale
